## 使用 C 實作具有 增、刪、查、改 操作的 Link List，並使用 CommandLine 進行操作。

### 操作介面
```bash
輸入節點數量: 3
輸入節點 1 資料值: 10
輸入節點 2 資料值: 20
輸入節點 3 資料值: 30
----------
 'q': 退出
-----
 'i': 插入節點
-----
 'f': 新增首節點
-----
 'd': 刪除節點
-----
 'c': 變更節點
-----
 'r': 翻轉 Link List
-----
 's': 遍歷節點
-----
輸入要執行的指令: s
--> 10 20 30
```

### 使用方法
1. clone 下來
2. 使用 `gcc -o main mian.c linklist.c` 進行編譯
3. 執行 `main` 執行檔
